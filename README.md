# Python 3 library for https://bitmix.biz / http://bitmixbizymuphkc.onion

## Installation

* `pip3 install bitmix || pip install bitmix`

## Usage

* `bitmix mix --currency bitcoin --address 1a....`

* `bitmix check (ID from previous output)`
